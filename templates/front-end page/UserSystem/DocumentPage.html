Time series Stock Forecast API

API Parameters

Required: apikey

You need an API Key to use our service, you can get a free one by clicking the link below. In this case, apikey=demo


Require: symbol

You need to enter a correct and existing ticker symbol for the system to correctly predict the stock price, symbol=AAPL


Optional: apikey

Stock prediction is based on neural network model, we provide default linear regression for stock prediction, you can also fill in our supported models [lr,lstm,gru] for prediction, model=lr


Optional: timeframe

You need to enter the number of days they want to predict, we have provided a default value of 5 days with a top line of 100. Be aware that the longer the number of days, the less accurate the stock prediction will be timeframe=5


Optional: layers

Layers define the number of hidden layers in the neural network, appropriate values will allow the model to fit the best function and produce better results, but too small or too large a value will have an impact on the accuracy of the results, we provide the default value of 3 layers. layers=3


Optional: neurons

neurons determine the number of neurons contained in each hidden layer, more neurons can come to capture complex patterns and features in the data, we provide the default number of 16 neurons. neurons=16
